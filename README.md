# AI 职场提效课程项目

这是一个围绕“普通职场人如何真正把 AI 用进工作里”的课程制作仓库，用来管理课程结构、录课 Markdown、逐字稿以及阶段性素材。

## 项目目标

这门课不是单纯介绍工具，而是帮助学习者建立一套更贴近真实工作的 AI 使用方式，包括：

- 理解 AI 时代的工作变化
- 搭建最小可用的 AI 工作环境
- 提升信息处理、写作表达、分析判断、沟通协作等核心能力
- 形成可复用的个人 AI 工作流
- 通过实战把 AI 变成稳定的工作协作者

## 课程结构

当前课程已确定 10 个大章节：

1. AI时代，如何跟上时代发展
2. 开始用AI工作前，要先准备什么
3. 信息处理提效
4. 写作表达提效
5. 分析判断提效
6. 沟通协作提效
7. 岗位场景应用
8. 建立个人AI工作流
9. Vibe Coding实战
10. 综合实战

## 当前工作流

课程制作流程已经固定为：

1. 先确定章节和小节结构
2. 先写录课用 Markdown 文档
3. 讨论并优化 Markdown 的信息密度和展示方式
4. 再写逐字稿
5. 用户确认后再保存为 `.txt`

## 文档风格约定

- Markdown 是给观众看的，不是逐字稿
- Markdown 内容要精简，避免像照着文档念
- 标题层级不过度拆分，按讲课节奏使用
- 不使用 `---` 作为分隔线
- 逐字稿保持纯口播文本，方便录音和字幕处理

## 仓库内容说明

当前仓库主要包含：

- 已完成小节的逐字稿 `.txt`
- 录课辅助用的 Markdown 文档
- 阶段性回看或复盘资料目录

后续可以继续补充：
